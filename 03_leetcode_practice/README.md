# 第四阶段：LeetCode 实战

> 🎯 目标：通过 Hot 100 高频题，达到 Medium 难度稳定通过
> ⏱️ 预计时间：2-3 周

## 📌 练习原则

1. **先自己做**：每题先独立思考 15-30 分钟
2. **限时训练**：Easy 15分钟，Medium 30分钟，Hard 45分钟
3. **理解思路**：看题解时理解思路，不要死记代码
4. **重复练习**：做错的题 3 天后重做
5. **归纳总结**：每做完一类题，总结解题模板
